30. ~MathJawab soal dengan lengkap! ​


Diberikan persamaan linear:a. y + 2x = 6b. 3y + 4x - 5 = 0Kemudian, tentukan pasangan berurutan dengan tabel dan gambarkan grafik persamaan fungsi tersebut!

______________________________

JAWABAN:

a. (3, 0) dan (0, 6), tabel dan grafik fungsi terlampir.b. (5/4, 0) dan (0, 5/3), tabel dan grafik fungsi terlampir.Gambar secara berurut = { Tabel a, Tabel b, Grafik Fungsi a, Grafik Fungsi b, Grafik Fungsi Gabungan a dan b }

PENJELASAN:

⇔ Sistem Persamaan Linear Dua Variabel (SPLDV) adalah sistem persamaan linear yang hanya memiliki/memuat dua variabel dalam ruas dengan pangkat satu.

⇔ SPLDV dapat diselesaikan dengan metode substitusi, eliminasi, grafik, Gauss-Jordan (matriks) dan gabungan diantaranya.

⇔ Grafik fungsi persamaan linear merupakan grafik fungsi tidak berlimit, namun mempunyai titik potong (intercept) x dan y yang membentang lurus.

⇔ Untuk menyelesaikan grafik persamaan linear dari suatu persamaan, kita dapat mensubstitusikan salah satu variabel (misal x atau y) dengan 0 dan mendapatkan koordinat salah satu variabel. Gunakan cara yang sama untuk variabel yang lain, kemudian lukis dalam grafik.

PEMBAHASAN SOAL:

⇒ a. y + 2x = 6

Terdapat dua variabel disini, ada variabel y dengan koefisien 1 dan x dengan koefisien 2 dengan nilai konstanta 6 di ruas kanan.

Selesaikan dengan mensubstitusikan salah satu variabel dengan 0 dan dapatkan pasangan koordinat (x, y).

→ Substitusikan nilai y = 0:

y + 2x = 60 + 2x = 62x = 62x : 2 = 6 : 2x = 3 dan y = 0(x, y) = (3, 0)

→ Substitusikan nilai x = 0:

y + 2x = 6y + 2(0) = 6y = 6 dan x = 0(x, y) = (0, 6)

Jadi, titik potong horizontal (garis x) adalah (3, 0) dan titik potong (intercept) vertikal (garis y) adalah (0, 6).

→ Tabel dan Grafik Fungsi Linear terlampir.

⇒ b. 3y + 4x - 5 = 0

Terdapat dua variabel disini, ada variabel y dengan koefisien 3 dan x dengan koefisien 4 dengan nilai konstanta -5 di ruas kiri dan 0 di ruas kanan.

Selesaikan dengan mensubstitusikan salah satu variabel dengan 0 dan dapatkan pasangan koordinat (x, y).

→ Substitusikan nilai y = 0:

3y + 4x - 5 = 03(0) + 4x - 5 + 5 = 0 + 54x = 54x : 4 = 5 : 4x = 5/4 dan y = 0(x, y) = (5/4, 0)

→ Substitusikan nilai x = 0:

3y + 4x - 5 = 03y + 4(0) - 5 + 5 = 0 + 53y = 53y : 3 = 5 : 3y = 5/3 dan x = 0(x, y) = (0, 5/3)

Jadi, titik potong horizontal (garis x) adalah (5/4, 0) dan titik potong (intercept) vertikal (garis y) adalah (0, 5/3).
