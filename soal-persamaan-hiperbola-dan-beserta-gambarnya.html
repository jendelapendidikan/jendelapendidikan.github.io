10. diberikan hiperbola dengan puncak (-2,3) dan (-2,9). jika puncak berada di tengah tengah antara pusat dan focus, maka persamaan hiperbola itu adalah?


Irisan Kerucut
Kelompok peminatan kelas XI kurikulum 2013 revisi 2016.

Jika kedua titik fokus dihubungkan dengan suatu garis, garisnya vertikal dan sejajar sumbu Y. Ini berati bentuk persamaan hiperbola tersebut adalah:
(y - k)² / a² - (x - h)² / b² = 1

Puncak ≡ (h, k ± a)
k - a = 3
k + a = 9
------------ -
-2a = -6 → a = 3
k + 3 = 9 → k = 6

Pusat ≡ (h, k) = (-2, 6)

Sketsa hubungan antar titik: (-2, 6) ----- (-2, 9) ----- (-2, y₁)
(-2, 9) sebagai titik tengah. Untuk titik fokus F₁:
(6 + y₁) / 2 = 9 → y₁ = 12

Sketsa hubungan antar titik: (-2, y₂) ----- (-2, 3) ----- (-2, 6)
(-2, 3) sebagai titik tengah. Untuk titik fokus F₂:
(y₂ + 6) / 2 = 3 → y₁ = 0

Fokus ≡ (h, k ± c)
6 + c = 12
6 - c = 0
Jelas terlihat bahwa c = 6.
Fokus = F₁(-2, 0) dan F₂(-2, 12)

c² = a² + b²
b² = c² - a² = 6² - 3² = 27

(y - k)² / a² - (x - h)² / b² = 1
(y - 6)² / 9 - (x + 2)² / 27 = 1

16. apa rumus persamaan hiperbola


Jawab:

[tex]\displaystyle \frac{x^2}{a^2}-\frac{y^2}{b^2}=1[/tex] Hiperbola horizontal berpusat di O(0, 0)

[tex]\displaystyle \frac{y^2}{a^2}-\frac{x^2}{b^2}=1[/tex] Hiperbola vertikal berpusat di O(0, 0)

[tex]\displaystyle \frac{(x-h)^2}{a^2}-\frac{(y-k)^2}{b^2}=1[/tex] Hiperbola horizontal berpusat di P(h, k)

[tex]\displaystyle \frac{(y-k)^2}{a^2}-\frac{(x-h)^2}{b^2}=1[/tex] Hiperbola vertikal berpusat di P(h, k)

Penjelasan dengan langkah-langkah:

Cara termudah menggambar hiperbola dengan 2 pin, pensil, tali tanur dan penggaris.

Ketentuan pada hiperbola [tex]\displaystyle c > a > b[/tex] dan [tex]\displaystyle b^2=c^2-a^2[/tex]

a merupakan ½ panjang sumbu mayor (panjang)

b merupakan ½ panjang sumbu minor (pendek)

c merupakan jarak titik pusat ke fokus

Sekarang mencari persamaan hiperbola horizontal yang berpusat di O(0, 0) berdasarkan gambar tadi.

• Ikat pin dan tali di ujung penggris dan ujung satunya pasang pin dan pin yang terpasang di ujung penggaris tandai sebagai tirik fokus pertama

• Gambar hiperbola dengan menaik turunkan pensil dan penggaris sampai menjadi seperti itu

• Dengan cara yang sama terbentuklah hiperbola

Jika (x, y) terletak di (a, 0) maka agar diperoleh 2a dengan melibatkan d₁ dan d₂

[tex]\displaystyle 2a+c-a-(c-a)=d_1-d_2\\d_1-d_2=2a[/tex]

Berdasarkan rumus jarak kedua titik

[tex]d_1-d_2=2a\\\sqrt{(x+c)^2+(y-0)^2}-\sqrt{(x-c)^2+(y-0)^2}=2a\\\sqrt{(x+c)^2+y^2}=2a+\sqrt{(x-c)^2+y^2}\\(x+c)^2+y^2=4a^2+4a\sqrt{(x-c)^2+y^2}+[(x-c)^2+y^2]\\x^2+2cx+c^2+y^2=4a^2+4a\sqrt{(x-c)^2+y^2}+x^2-2cx+c^2+y^2\\4cx=4a^2+4a\sqrt{(x-c)^2+y^2}\\a\sqrt{(x-c)^2+y^2}=cx-a^2[/tex]

[tex]\displaystyle a^2[(x-c)^2+y^2]=c^2x^2-2a^2cx+a^4\\a^2(x^2-2cx+c^2+y^2)=c^2x^2-2a^2cx+a^4\\a^2x^2-2a^2cx+a^2c^2+a^2y^2=c^2x^2-2a^2cx+a^4\\a^2(c^2-a^2)=(c^2-a^2)x^2-a^2y^2\\a^2b^2=b^2x^2-a^2y^2\\\boxed{\frac{x^2}{a^2}-\frac{y^2}{b^2}=1}[/tex]

Dengan merotasikan 90° terhadap titik O(0, 0) untuk hiperbola vertikal yang berpusat di O(0, 0)

[tex]\displaystyle (x,y)\xrightarrow[]{R(O,90^\circ)}(-y,x)\\x'=-y\\y'=x\\\frac{x^2}{a^2}-\frac{y^2}{b^2}=1\\\frac{(-y')^2}{a^2}-\frac{(x')^2}{b^2}=1\\\boxed{\frac{y^2}{a^2}-\frac{x^2}{b^2}=1}[/tex]

Dengan konsep translasi jika hiperbola horizontal pusat nya di P(h, k)

[tex]\displaystyle (x,y)\xrightarrow[]{T\binom{h}{k}}(x+h,y+k)\\x'=x+h\\y'=y+k\\\frac{x^2}{a^2}-\frac{y^2}{b^2}=1\\\frac{(x'-k)^2}{a^2}-\frac{(y'-k)^2}{b^2}=1\\\boxed{\frac{(x-h)^2}{a^2}-\frac{(y-k)^2}{b^2}=1}[/tex]

Dengan konsep translasi pula jika elips vertikal pusat nya di P(h, k)

[tex]\displaystyle (x,y)\xrightarrow[]{T\binom{h}{k}}(x+h,y+k)\\x'=x+h\\y'=y+k\\\frac{y^2}{a^2}-\frac{x^2}{b^2}=1\\\frac{(y'-k)^2}{a^2}-\frac{(x'-h)^2}{b^2}=1\\\boxed{\frac{(y-k)^2}{a^2}-\frac{(x-h)^2}{b^2}=1}[/tex]

Fokus : q + c = -5 => 0 + c = -5 => c = -5
c^2 = a^2 + b^2

Perhatikan soal! Apabila titik pusat hiperbola, puncak dan fokus dihubungkan, maka sejajar sumbu Y yang berati persamaan hiperbola tersebut berbentuk:
(y - k)² / a² - (x - h)² / b² = 1

Titik pusat ≡ (h, k) = (1, 0)

Titik puncak ≡ (h, k ± a)
Salah satu titik puncaknya (1, 4).
k ± a = 4
0 ± a = 4 → a = 4 ∨ a = -4

Titik fokus ≡ (h, k ± c)
Salah satu titik fokusnya (1, -5).
k ± c = -5
0 ± c = -5 → c = 5 ∨ c = -5

a > b
c² = a² + b²
b = √(c² - a²) = √(5² - 4²) = 3
Persamaan hiperbolanya:
(y - k)² / a² - (x - h)² / b² = 1
(y - 0)² / 4² - (x - 1)² / 3² = 1
y² / 16 - (x - 1)² / 9 = 1

Maka persamaan asimtotnya:
y = ±a/b (x - h) + k
   = ±4/3 (x - 1) + 0
y = 4/3 (x - 1) + 0 dan y = -4/3 (x - 1) + 0
4x - 3y - 4 = 0 dan 4x + 3y - 4 = 0

27. Gambarlah hiperbola untuk persamaan 16x² - 9y² - 64x + 18y - 89 = 0


Penjelasan dengan langkah-langkah:

Untuk menggambar hiperbola dengan persamaan 16x² - 9y² - 64x + 18y - 89 = 0, kita perlu mengubahnya menjadi bentuk standar hiperbola. Bentuk standar hiperbola umumnya ditulis sebagai (x-h)²/a² - (y-k)²/b² = 1 atau (y-k)²/b² - (x-h)²/a² = 1.

Mari kita ubah persamaan tersebut menjadi bentuk standar dengan melengkapi kuadrat sempurna pada variabel x dan y.

16x² - 9y² - 64x + 18y - 89 = 0

Mengelompokkan suku-suku yang serupa:

(16x² - 64x) - (9y² - 18y) = 89

Memfaktorkan suku-suku kuadrat:

16(x² - 4x) - 9(y² - 2y) = 89

Melengkapi kuadrat sempurna pada x dengan (x - 2)² = x² - 4x + 4, dan pada y dengan (y - 1)² = y² - 2y + 1:

16(x² - 4x + 4) - 9(y² - 2y + 1) = 89 + 64 - 9

16(x - 2)² - 9(y - 1)² = 144

Divide both sides by 144 to simplify:

(x - 2)²/9 - (y - 1)²/16 = 1

Sekarang kita telah mendapatkan bentuk standar hiperbola. Dari persamaan tersebut, kita dapat melihat bahwa hiperbola memiliki pusat di titik (2, 1), sumbu utama sejajar dengan sumbu x, dan sumbu pendek sejajar dengan sumbu y.

Untuk menggambar hiperbola ini, kita dapat menggunakan pusat (2, 1), serta a = 3 dan b = 4 (dari akar kuadrat denominators pada persamaan standar).

Dengan menggunakan informasi ini, kita dapat menggambar dua lengkungan yang berfokus pada pusat (2, 1) dengan panjang sumbu utama sejajar dengan sumbu x sebesar 2a = 6 (titik ujung horisontal) dan panjang sumbu pendek sejajar dengan sumbu y sebesar 2b = 8 (titik ujung vertikal).

Jadi, hiperbolanya terletak di sekitar pusat (2, 1), dengan sumbu utama sejajar dengan sumbu x dan sumbu pendek sejajar dengan sumbu y.

31. Diketahui persamaan hiperbola adalah 16x² - 9y² - 32x + 18y - 137 = 0. a. Tentukan pusat hiperbola? b. Persamaan asimtotnya? c. Gambarkan grafik fungsinya!​


Jawaban:

Untuk menentukan pusat hiperbola dari persamaan yang diberikan, kita perlu melengkapi persamaan menjadi bentuk kanonik. Dalam bentuk kanonik, persamaan hiperbola memiliki rumus sebagai berikut:

((x - h)² / a²) - ((y - k)² / b²) = 1

Dalam persamaan tersebut, (h, k) adalah pusat hiperbola, dan a dan b adalah panjang setengah sumbu utama dan setengah sumbu pendek hiperbola.

Mari kita lakukan penyelesaian untuk mencari pusat hiperbola:

16x² - 9y² - 32x + 18y - 137 = 0

Dalam hal ini, kita perlu melengkapi persamaan dengan melengkapi kuadrat untuk x dan y. Jadi kita dapat mengubah persamaan menjadi:

16(x² - 2x) - 9(y² - 2y) = 137

16(x² - 2x + 1) - 9(y² - 2y + 1) = 137 + 16 - 9

16(x - 1)² - 9(y - 1)² = 144

Sekarang kita dapat membandingkan persamaan dengan rumus kanonik:

((x - 1)² / 3²) - ((y - 1)² / 4²) = 1

Dari sini, kita bisa melihat bahwa pusat hiperbola berada di titik (1, 1).

Untuk mencari persamaan asimtotnya, kita dapat menggunakan rumus:

y = ± (b/a) * (x - h) + k

Dalam hal ini, (h, k) adalah pusat hiperbola, dan a dan b adalah panjang setengah sumbu utama dan setengah sumbu pendek hiperbola.

Dalam kasus ini, pusat hiperbola adalah (1, 1), dan panjang setengah sumbu utama adalah 4, sedangkan panjang setengah sumbu pendek adalah 3.

Sehingga persamaan asimtotnya menjadi:

y = ± (4/3) * (x - 1) + 1

y = (4/3)x - (4/3) + 1

y = (4/3)x - 1/3

y = (4/3)x + 2/3

y = -(4/3)x + 5/3

Untuk menggambarkan grafik fungsinya, kita bisa menggunakan pusat hiperbola dan persamaan asimtotnya. Berdasarkan persamaan asimtotnya, kita tahu bahwa hiperbola akan cenderung mendekati garis dengan kemiringan (gradien) sekitar ±4/3 saat jauh dari pusat.

Grafik hiperbola akan membuka ke atas dan ke bawah, dan asimtotnya akan menjadi batasnya. Oleh karena itu, kita dapat menggambarkan dua garis asimtot dengan gradien ±4/3 dan mencari titik-titik di sekitarnya untuk menggambarkan kurva hiperbola.

Berikut adalah gambaran grafik hiperbola dengan pusat (1, 1) dan persamaan asimtotnya
