11. Apakah diantara dua bilangan real yang berdekatan ada bilangan real lainnya, kalau ada buktikan!


Saya gunakan metode pembuktian dengan kontradiksi.

Pertama, saya asumsikan bahwa di antara dua bilangan real hanya ada bilangan tidak real.
Jadikan A,B dua bilangan real.
Lalu kita tahu bahwa (A+B)/2 hasilnya selalu di antara A dan B, yakni A<x<B di mana x adalah hasil dari (A+B)/2. Namun x disini ternyata adalah bilangan real. Maka asumsi awal salah.
Sehingga, terbukti bahwa antara dua bilangan real ada bilangan real lainnya

22. soal nya Fungsi n dari Himpunan Bilangan Real R ke Himpunan Bilangan Real R didefinisikan dengan grafik sebagai berikut dan itu gambar grafik nya tolong jawab kan


Fungsi n dari himpunan bilangan real ke himpunan bilangan real didefinisikan dengan grafik seperti tampak pada gambar.
a. Fungsi n dinyatakan dengan cara pasangan berurutan adalah sebagai berikut.
Karena n : R → R, maka setiap titik yang ada pada kurva parabola adalah anggota dari himpunan pasangan berurutan n. Sebagai representasinya kita bisa mengambil beberapa titik yang jelas terlewati kurva meskipun tidak mengetahui rumus fungsi dari kurva.
Titik-titk tersebut antara lain (-10,0), (0,-20), (10,0).
Sehingga kita dapat menuliskan
n = {…, (-10,0), (0,-20), (10,0), …}
poin b dan c terlampir.

Setiap penyajian sebuah relasi/fungsi memiliki kelebihan dan kekurangan masing-masing.
Fungsi yang sudah diketahui rumusannya dan memiliki domain serta range himpunan bilangan real akan lebih representatif jika disajikan dengan kurva, karena kurva bisa menggambarkan setiap titik yang dipetakan oleh fungsi.
Penyajian dalam bentuk diagram dan tabel lebih tepat digunakan untuk fungsi dengan domain dan range yang terbatas atau jumlahnya sedikit.

24. contoh bilangan real dan rasional beserta penjelasan


1. bilangan real adalah bilangan yang merupakan gabungan dari bilangan rasional dan irrasional sendiri.
cont : 0, 1, 2 1/2, 4/7, 55/7, √2, √3, √5... dst

2. bilangan rasional adalah bilangan dalam bentuk a/b, dengan a dan b anggota dari bilangan bulat dan b ≠ 0
cont : 1/4, menjadi a =1 dan b=4
Himpunan bilangan yang paling luas merupakan Himpunan Bilangan Kompleks yang terdiri dari :
1. Himpunan Bilangan Imajiner (Bilangan Khayal)
Contoh : [tex] \sqrt{-1}[/tex]
2. Himpunan Bilangan Real (Bilangan Nyata)
Contoh : 1, 0,5, - 7, [tex] \sqrt{2}, \frac{1}{5}[/tex]
Himpunan bilangan Real sendiri terdiri dari :
1. Himpunan Bilangan Irasional
Himpunan Bilangan Irasional merupakan bilangan yang tidak bisa dinyatakan dalam bentuk [tex] \frac{a}{b}[/tex] dengan a dan b merupakan bilangan bulat dengan
Contoh : [tex] \sqrt{2}[/tex]
2. Himpunan Bilangan Rasional
Himpunan Bilangan Rasional Merupakan bilangan yang bisa dinyatakan dalam bentuk [tex] \frac{a}{b}[/tex] dengan a dan b merupakan bilangan bulat.
Contoh :
a. [tex] \frac{2}{5}[/tex]
b. [tex]0,676767...= \frac{67}{99}[/tex]
c. [tex]4= \frac{4}{1}= \frac{8}{2} [/tex]

25. 3 soal tentang banyaknya faktor real negatif dari suku banyak beserta pembahasannya

1. Salah satu faktor dari 2x³ - 5x² - px + 3 adalah ( x + 1 ). Faktor lain dari suku banyak tersebut adalah...

A. (x - 2) dan (x - 3)

B. (x + 2) dan (2x - 1)

C. (2x - 1) dan (x - 3)

D. (2x + 1) dan (x - 2)

Pembahasan :

Karena x + 1 merupakan faktor suku banyak, maka suku banyak habis dibagi dan berlaku f(-1) = 0.

f(x) = 2x² - 5x² - px + 3 dari (x + 1) diperoleh x = -1

=> f(-1) = 2(-1)³ - 5(-1)² - p(-1) + 3

=> 0 = -2 - 5 + p + 3

=> 4 = p => substitusi nilai p = 4 ke fungsi suku banyak.

f(x) = 2x³ - 5x² - 4x + 3

=> f(x) = (x + 1) (2x² - 7x + 3)

=> f(x) = (x + 1) (2x - 1) (x - 3)

Jadi faktor lainnya adalah

C. (2x - 1) dan (x - 3)

37. apa yang dimaksud dengan bilangan real rasional dan bilangan real irrasional


Bilangan real rasional adalah bilangan yang dapat dinyatakan dalam a/b, dan a,b adalah rasional.

Misalnya:
1.) 3 merupakan real rasional karena 3 = 3/1
2.) 1/6 merupakan real rasional (jelas)
3.) 6,47 merupakan real rasional karena 6,47 = 647/100

Bilangan real irasional adalah bilangan real yang tidak dapat dinyatakan dalam a/b
Misalnya pada akar kuadrat:
1.) √3 tidak dapat dinyatakan dalam a/b
Pada konstanta irasional lainnya
2.) π tidak dapat dinyatakan dalam a/b
3.) ∛9 tidak dapat dinyatakan dalam a/b
bilangan rasional adalah bilangan yang dapat dinyatakan menjadi a/b
contohnya : 1/2 = 0,5
bilangan irasional adalah bilangan yang tidak dapat dinyatakan menjadi a/b
contohnya : √2 = 1,4........
√2 tidak dapat dikatakan rasional karena √2 tidak dapat diubah ke bentuk bilangan pasti
