1. Contoh soal pembiasan pada permukaan cembung beserta pembahasannya (Kelas XII)


Lima puluh centimeter di depan cermin cembung ditempatkan sebuah benda. Titik pusat kelengkungan cermin 50 cm. Tentukan jarak bayangan ke cermin dan perbesaran bayangan itu!


Jawab:

Diketahui:

s = 50 cm

R = - 50 cm


Ditanyakan:

s' = ?

M = ?


Penyelesian:

R = 2f => f = R/2 = - 50 cm/2 = - 25 cm

1/f = 1/s + 1/s’

1/s’ = 1/f – 1/s

1/s’ = (1/-25) – 1/50

1/s’ = -2/50 – 1/50

1/s’ = -3/50

s' = - 50/3  

s’ = - 16,67 cm


M = s’/s = 16,67 cm/50 cm

M = 0,33


Jadi, jarak bayangan ke cermin adalah 16,67 cm dan perbesaran bayangannya adalah 0,33 kali.

20. SOAL PRODUKTIFI KELAS XI TKJ (A-D) 1. Jelaskan yang di maksud dengan web server, dan berikan 2 contoh serta jelaskan web server yang banyak digunakan pada saat ini! 2. Sebutkan 3 contoh aplikasi Web Browser dan jelaskan masing-masing 3 contoh? 3. sebutkan 3 contoh bahasa pemrograman yang digunakan untuk membuat sebuah web. Kemudian jelaskan cara kerja sebuah web server! 4. Jelaskan apa yang dimaksud dengan DNS Server! 5. Sebutkan dan jelaskan fungsi dari DNS Server!​


Jawaban:

1. Web server adalah sebuah program atau software yang berfungsi untuk menerima dan memproses permintaan dari browser web untuk menampilkan halaman web.

Dua contoh web server yang banyak digunakan saat ini adalah Apache dan Nginx. Apache adalah web server open-source yang sangat populer dan digunakan oleh sebagian besar situs web di seluruh dunia. Nginx adalah web server open-source yang dirancang untuk menangani lalu lintas web dengan efisien pada server modern yang sibuk.

2. Tiga contoh aplikasi web browser yang populer adalah:

Google Chrome: aplikasi web browser yang dikembangkan oleh Google. Fiturnya meliputi tab browsing, penjelajahan pribadi, dan integrasi dengan Google Drive dan aplikasi Google lainnya.Mozilla Firefox: aplikasi web browser open-source yang populer dengan fitur-fitur seperti tab browsing, penjelajahan pribadi, dan dukungan untuk add-on dan ekstensi.Microsoft Edge: aplikasi web browser yang dikembangkan oleh Microsoft. Fiturnya meliputi tab browsing, integrasi dengan Microsoft Office dan OneDrive, dan mode penghematan baterai untuk perangkat mobile.

3. Tiga contoh bahasa pemrograman yang digunakan untuk membuat web adalah:

HTML (Hypertext Markup Language): bahasa markah yang digunakan untuk membuat struktur dan konten halaman web.CSS (Cascading Style Sheets): bahasa stylesheet yang digunakan untuk mengatur tampilan dan format halaman web.JavaScript: bahasa scripting yang digunakan untuk membuat interaksi dan efek di halaman web.

Web server bekerja dengan menerima permintaan dari browser web melalui protokol HTTP (Hypertext Transfer Protocol) atau HTTPS (Hypertext Transfer Protocol Secure). Setelah menerima permintaan, server memproses permintaan tersebut dan mengirimkan kembali halaman web yang diminta ke browser.

4. DNS Server (Domain Name System Server) adalah sebuah server yang berfungsi untuk mengubah nama domain (seperti google.com) menjadi alamat IP yang sesuai (seperti 172.217.168.46) sehingga dapat diakses oleh komputer melalui jaringan internet.

5. Fungsi dari DNS Server adalah sebagai berikut:

Menerjemahkan nama domain menjadi alamat IP yang sesuai.Mengelola cache DNS untuk meningkatkan kecepatan akses ke situs web yang sering dikunjungi.Mencegah serangan DNS Spoofing atau DNS Cache Poisoning dengan memastikan bahwa alamat IP yang dikembalikan benar-benar berasal dari server DNS yang sah.

A. 41
B. 50
C, 59
D. 70
E. 82

Pembahasan
Diketahui:
Sin Ik = 0,6 = 6 / 10
Ditanya: Ip = ....
Jawab:
tan Ip = n2 / n1 = 10 / 6 = 1,67
Ip = 59
Jawaban: C

Cahaya matahari jatuh permukaan air yang indeks biasnya 4/3 sehingga menghasilkan cahaya pantul terpolarisasi linear. Hitunglah:
a. Sudut polarisasi
b. Sudut bias

Pembahasan
Diketahui:
n2 = 4/3
n1 = 1
Ditanya: Ip dan r
Jawab:
Menghitung Ip
tan Ip = n2/n1 = 4/3 / 1 = 4/3 = 1,33
Ip = arc tan 1,33 = 53
Menghitung sudut bias
n1 sin Ip = n2 cos r
1 sin 53 = 1,33 cos r
0,8 = 4/3 cos r
cos r = 0,067
r = arc cos 0,067

25. Carilah informasi tentang pemrograman web​


Jawaban: Pemrograman Web

Penjelasan:

adalah proses atau cara untuk menjalankan intruks Pada sebuah komputer yang terhubung ke internet untuk membuat fungsi atau tugas tertentu

Jawaban dan Penjelasan:

Pemrograman web, atau web programming, adalah aktivitas menulis kode-kode program dalam proses atau tahapan pengembangan/pembangunan web (web development), yang di antaranya meliputi konten web, scripting web dari sisi client dan server, serta keamanan jaringan, terkait dengan web yang dibangun.

Pemrograman web berbeda dengan hanya “memprogram” atau “membuat program”, karena memerlukan pengetahuan interdisipliner (antara beberapa disiplin ilmu yang berhubungan) pada area aplikasi, scripting di sisi client dan server, serta teknologi basisdata.

Oleh karena itu, secara garis besar, terdapat dua buah kategorisasi pemrograman web, yaitu:

Pemrograman web pada sisi client
Pada sisi client, pemrograman diperlukan dalam hal pengaksesan data dan penyajian informasi dari dan kepada pengguna web.
Pemrograman web pada sisi client juga diperlukan untuk meningkatkan pengalaman pengguna (user experience) dalam berinteraksi dengan web yang dikunjungi.
Untuk hal ini, seorang web programmer setidaknya harus menguasai pengolahan DOM (Document Object Model), HTML, XML, Javascript, dan CSS. Pemrograman web pada sisi server
Pada sisi server, pemrograman terutama diperlukan dalam hal pengambilan data (dari database), keamanan jaringan dan website, serta kinerja web.
Bahasa pemrograman yang dapat digunakan adalah PHP, ASP, Java, Python, dan lain-lain.
Untuk database, yang umum digunakan adalah MySQL atau MariaDb, namun tidak jarang juga yang menggunakan MS SQL Server dan PostgreSQL, atau bahkan file teks biasa yang terorganisir dan sistematis.
