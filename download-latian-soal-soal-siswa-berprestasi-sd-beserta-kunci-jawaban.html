9. Dalam tes siswa berprestasi yang terdiri dari 40 soal peserta mendapat skor 3 untuk jawaban benar skor-2 untuk setiap jawaban yang salah dan skor-1 untuk soal yang tidak dijawab. Jika Rizal menjawab 36 soal dan memperoleh skor 64, banyak soal yang dijawab benar oleh Rizal adalah​


Banyak soal yang di jawab benar oleh rizal apabila rizal menjawab 36 soal dan memperoleh skor 64 adalah 28 soal.

Penjelasan dengan langkah-langkah:

Diketahui:

- Riza menjawab 36 soal (total dari soal yang benar dijawab (a) dan yang salah (b)) → Perumpamaan : a+b = 36

- Soal benar bernilai +3 → 3 x a

- Soal salah bernilai  -2 → -2 x b

- Soal tidak terjawab bernilai -1 → -1 x c

Ditanya:

Banyak soal yang dijawab benar adalah? atau nilai a = ?

Dijawab:

Skor yang didapat = (3 x jawab benar) + (-2 x jawab salah) + (-1 x tidak jawab)

Skor yang didapat = (3 x a) + (-2 x b) + (-1 x c) → nilai b : a + b = 36 → b = 36-a

                                                                          → nilai c : c = total soal - dijawab

                                                                                         c = 40 - 36 = 4

Skor yang didapat = (3 x a) + (-2 x (36-a)) + (-1 x 4)

            64             = 3a + (-72 + 2a) + (-4)

            64             = 3a - 72 + 2a - 4

            64             = 5a - 76

            5a             = 64 + 76

            5a             = 140

             a              = 28  → nilai a adalah jumlah soal yang dijawab benar

b = 36 - a

b = 36 - 28

b  = 8  → nilai b adalah jumlah soal yang dijawab salah

Maka, soal yang dijawab benar oleh Rizal sebanyak 28 soal.

Skor tidak menjawab = 4 × -1 = -4

Jumlah soal benar = x

Jumlah soal salah = y

3x - 2y - 4 = 64

3x - 2y = 68

y = 36 - x

3x - 2(36 - x) = 68

3x - 72 + 2x = 68

5x = 140

x = 28

Jumlah soal benar = 28
