Title (*): Isi dengan Mr (Bapak), Mrs (Ibu), Ms, atau Miss (Nona). First (Given) Name (*): Nama Depan Middle Name: Nama Tengah (dapat dikosongkan bila nama anda kurang dari 3 kata). Bila nama anda 2 kata boleh mengisi First (Given) dan Last (Sur) Name saja. Sedangkan bila 1 kata, sebagai contoh Donny. Maka anda dapat mengisi Donny pada First (Given) dan Donny pada Last (Sur) Name. Last (Sur) Name (*): Nama Belakang / Marga Citizenship (*): Kewarganegaraan / nationality. Contoh : anda warga negara Indonesia meskipun anda tinggal di luar negeri maka Citizenship tetap diisi dengan Indonesia. Country of Residence (*): Negara domisili yang ditinggali saat ini. Contoh : anda warga negara Indonesia tetapi saat ini tinggal di Jepang, maka Country of Residence dapat diisi Jepang. ID Type (*): Tipe Dokumen Identitas yang akan anda upload untuk verifikasi Account Trading. Ada 3 pilihan yaitu : Passport, National ID Card (KTP / MyKad), dan Driver's License (SIM / Lesen Memandu). Identification Number (e.g. Driver's License or Passport Number) (*): Isikan Nomer ID sesuai ID Type di atas yang akan anda upload. Bila anda menggunakan KTP isikan nomer KTP anda disini. Sedangkan bila menggunakan SIM isikan nomer SIM anda. Date of Birth (*): Isikan tanggal lahir sesuai format yang ada. Contoh (dd/mm/yyyy) dapat anda isi tanggal/bulan/tahun. Driver's License Number: Tidak perlu anda isi karena otomatis akan terisi jika anda memilih ID Type Driver's License dan telah mengisi kolom Identification Number dengan nomer Driver's License anda Gender (*): Pilih Male untuk Pria atau Female untuk Wanita. Gender harus sesuai jenis kelamin yang tercantum pada Passport / ID card / Driver License. Marital Status (*): Status Pernikahan. Married (Kawin) atau Single (Tidak Kawin) Number of Dependents: Jumlah anggota keluarga yang ditanggung. Contoh jumlah anak. Kolom ini dapat dikosongkan Forex Trading Business Residential Address (P.O. Box Addresses are not acceptable) (*): Alamat Domisili saat ini. Jangan mengisi dengan alamat PO Box atau alamat office. City/Town (*): Kota domisili saat ini. County/Province (*): Negara Bagian atau Provinsi. Contoh : Jawa Barat, Jawa Timur, DKI Jakarta, dll. Ingat County adalah negara bagian bukan negara (country) ! Postal Code (*): Kode Pos dapat diisi dengan 0000 bila anda tidak tahu kode pos anda. Time at Current Address (*): Lama tinggal di alamat sekarang. Isi dalam Tahun (Year) dan Bulan (Month). Contoh : 5 Year 3 Month (5 tahun 3 bulan). Bila anda telah tinggal di alamat domisili saat ini kurang dari 3 tahun, maka anda diharuskan mengisi negara dan alamat tinggal sebelumnya. Telephone (*) (country code + area code + number) : Isikan Nomer Telepon rumah yang dapat dihubungi. Formatnya adalah kode negara (otomatis terisi sesuai dengan kode negara dari kolom "Country of Residence" yang anda isi di atas) contoh : Indonesia akan terisi 62. Kode area dapat diisi 21 (tanpa 0) untuk kode area jakarta, atau 31 untuk Surabaya. Mobile Telephone (country code + area code + number) : Isikan Nomer Telepon Selluler yang dapat dihubungi. Untuk kode negara dapat anda isi dengan kode negara contoh Indonesia 62. Kode area dapat anda isikan kode provider contoh 81, 85, 83 (tanpa 0). Email Address (*): Isikan alamat email yang anda gunakan. Pastikan email anda valid dan tidak penuh ! Selalu usahakan untuk mengecek email secara periodik dan mengecek pada folder bulk / spam untuk pemberitahuan penting. Bila anda mendapati email dari perusahaan FXCorporate masuk ke dalam folder spam. Maka silahkan ditandai / mark dengan "Not Spam" Disarankan untuk menggunakan email yang quotanya besar / unlimited seperti yahoo atau gmail. Hindari menggunakan email telkom, atau email provider lokal yang quotanya kecil. Confirm Email Address (*): Ketik ulang alamat email di atas sekali lagi untuk konfirmasi. Klik "Continue"
