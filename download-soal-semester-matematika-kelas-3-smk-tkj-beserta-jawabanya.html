14. Tuliskan apa saja yang akan dipelajari di mapel desain grafis untuk smk tkj kelas 10​


Jawaban:

1. Dasar Desain

Dasar desain atau design fundamentals sangat penting untuk dipelajari. Mulai dari elemen hingga prinsip desain. Mulai dari titik, bentuk, warna, tekstur, hingga ruang dan garis perlu untuk dipelajari dan akan banyak dijumpai dalam desain. Dari sana kita juga akan belajar mengenai keseimbangan, kesatuan, pengulangan, kontras, proporsi, penekanan, hingga ritme. Hal ini harus dipelajari sehingga kita memiliki keahlian mendesain yang mampu menciptakan desain menarik dan mudah dipahami orang banyak.

2. Sejarah Desain dan Seni

Sejarah desain dan seni penting untuk dipelajari. Karena desain grafis sendiri memiliki gaya yang berbeda di setiap masanya. Hal ini tercermin dari adanya gaya pop, retro, klasik, modern, dan lainnya. Dengan mempelajari hal ini, desainer bisa membuat desain yang sesuai dengan konteks dan target pasar yang dituju.

3. Menggambar

Sedikit banyak, desain grafis adalah soal menggambar. Sehingga tidak heran ketika banyak desainer grafis yang juga ahli dalam menggambar. Mereka yang tidak ahli dalam menggambar pun biasanya akan bisa untuk membuat sketsa, sehingga mereka bisa memiliki visual akan apa yang akan mereka desain.

4. Fotografi  

Jangan pikir dengan mendesain kita tidak perlu memahami apapun soal fotografi. Karena fotografi juga merupakan bagian tidak terpisahkan dalam mendesain. Desainer yang baik akan memahami teknis fotografi mulai dari framing, warna, pencahayaan, fokus, hingga beragam teori fotografi lainnya. Semua hal ini akan sangat berguna ketika kita membuat sebuah desain grafis yang medianya menggunakan foto.

5. Tipografi

Tipografi adalah hal yang sangat penting untuk dipelajari desainer grafis. Bagaimana tidak, desain grafis biasanya akan memiliki tipografi untuk menyampaikan informasi. Teks sendiri juga penting bagi pembaca untuk memahami desain grafis. Karena desain grafis bukanlah lukisan yang bisa dinikmati hanya dengan gambar saja.

6. Desktop Publishing

Setelah Single Page Desain, kini desainer juga tampaknya perlu belajar mengenai desktop publishing. Di dalamnya termasuk mempelajari master page, style sheet, dan juga pewarnaan dan beragam hal lainnya. Layout juga merupakan hal penting yang perlu dipejalari dalam desktop publishing, sehingga pembaca akan bisa membaca dengan mudah pesan yang ada dalam desain.

7. User Interface

Kini, seiring berkembangnya zaman, user interface adalah hal yang penting untuk dipelajari. Apalagi jika bicara user interface digital. Proses ini akan bisa membuat desain visual berjalan secara beriringan dengan teknis dan sistem lainnya, sehingga pembaca tidak hanya mendapatkan pesan, tapi juga bisa berinteraksi langsung dengan desain.

8. Presentasi

Jangan salah, sebaiknya desainer juga bisa berkomunikasi dengan baik dan bisa presentasi dengan lancar. Hal ini lebih kepada bagaimana desainer bisa menyampaikan ide. Desainer yang bisa menyampaikan idenya dengan artikulatif akan bisa hadir dengan konsep desain yang matang.

9. Single Page Desain

Digital Publishing akan bisa diekplor dengan mempelajari prinsip-prinsip dasar desain dan mengaplikasikannya pada tata letak desain. Layouting dalam satu halaman bisa dijadikan keahlian yang penting untuk dipelajari desainer.

24. Buatlah 3 soal induksi matematika berserta jawabanya


Buatlah 3 soal induksi matematika berserta jawabannya

Buktikan bahwa: 1 + 2 + 3 + .... + n = ½ n (n + 1), untuk setiap n bilangan asli Buktikan pernyataan P(n) = n(n + 1)(n + 5) adalah bilangan kelipatan 3, untuk setiap n bilangan asli Buktikan bahwa 5ⁿ – 1 habis dibagi 4, untuk setiap n bilangan asli

Ada dua langkah dalam induksi matematika yaitu:

Buktikan bahwa untuk n = 1 benar Dengan mengasumsikan bahwa untuk n = k benar, maka buktikan bahwa untuk n = k + 1 juga benar

Pembahasan

1. Buktikan bahwa: 1 + 2 + 3 + .... + n = ½ n (n + 1), untuk setiap n bilangan asli

Pembuktian

Akan dibuktikan untuk n = 1 benar

n = ½ n (n + 1)

1 = ½ . 1 . (1 + 1)

1 = ½ . 1 . 2

1 = 1

(BENAR)

Jika untuk n = k benar yaitu

1 + 2 + 3 + .... + k = ½ k (k + 1)

Akan dibuktikan untuk n = (k + 1) juga benar

1 + 2 + 3 + … + k + (k + 1) = ½ (k + 1) ((k + 1) + 1)

|___________|

        ½ k (k + 1)   + (k + 1) = ½ (k + 1) (k + 2)

      ½ k (k + 1) + 1 (k + 1) = ½ (k + 1) (k + 2)

        (k + 1) (½ k + 1)         = ½ (k + 1) (k + 2)

        (k + 1) . ½ (k + 2)      = ½ (k + 1) (k + 2)

    ½ (k + 1) (k + 2)            = ½ (k + 1) (k + 2)

             |____________________|

                          Terbukti benar

Jadi terbukti bahwa 1 + 2 + 3 + .... + n = ½ n (n + 1), untuk setiap n bilangan asli

2. Buktikan pernyataan P(n) = n(n + 1)(n + 5) adalah bilangan kelipatan 3, untuk setiap n bilangan asli

Pembuktian

Akan dibuktikan untuk n = 1 benar

P(n) = n(n + 1)(n + 5)

P(1) = 1(1 + 1)(1 + 5)

P(1) = 1(2)(6)

P(1) = 12

(benar bahwa 12 kelipatan dari 3)

Jika untuk n = k benar yaitu

P(k) = k(k + 1)(k + 5) adalah bilangan kelipatan 3

Akan dibuktikan untuk n = (k + 1) juga benar

P(k + 1) = (k + 1) ((k + 1) + 1) ((k + 1) + 5)

P(k + 1) = (k + 1) (k + 2) (k + 6)

P(k + 1) = (k + 1) (k² + 6k + 2k + 12)

P(k + 1) = (k + 1) (k² + 8k + 12)

P(k + 1) = (k + 1) (k² + 5k + 3k + 12)

P(k + 1) = (k + 1) ((k² + 5k) + (3k + 12))

P(k + 1) = (k + 1)(k² + 5k) + (k + 1)(3k + 12)

P(k + 1) = (k + 1)k(k + 5) + (k + 1)3(k + 4)

P(k + 1) = k(k + 1)(k + 5) + 3(k + 1)(k + 4)

k(k + 1)(k + 5) adalah bilangan kelipatan 3 (berdasarkan n = k) 3(k + 1)(k + 4) sudah jelas merupakan bilangan kelipatan 3

Jadi

P(k + 1) = k(k + 1)(k + 5) + 3(k + 1)(k + 4) juga merupakan bilangan kelipatan 3

Jadi terbukti bahwa P(n) = n(n + 1)(n + 5) adalah bilangan kelipatan 3, untuk setiap n bilangan asli

3. Buktikan bahwa 5ⁿ – 1 habis dibagi 4, untuk setiap n bilangan asli

Pembuktian

Akan dibuktikan untuk n = 1 benar

5ⁿ – 1

= 5¹ – 1

= 5 – 1

= 4

(benar bahwa 4 habis dibagi 4ᵏ)

Jika untuk n = k benar yaitu

5ᵏ – 1 habis dibagi 4

Akan dibuktikan untuk n = (k + 1) juga benar

5ᵏ⁺¹ – 1

= 5ᵏ . 5¹ – 1

= 5ᵏ . 5 – 1

= 5ᵏ . (4 + 1) – 1

= 5ᵏ . 4 + 5ᵏ . 1 – 1

= 4 . 5ᵏ + 5ᵏ – 1

= (4 . 5ᵏ) + (5ᵏ – 1)

(4 . 5ᵏ) sudah jelas habis dibagi 4 (5ᵏ – 1) juga habis dibagi 4 (berdasarkan n = k)

Jadi (4 . 5ᵏ) + (5ᵏ – 1)  habis dibagi 4

Sehingga terbukti bahwa 5ⁿ – 1 habis dibagi 4, untuk setiap n bilangan asli

a. Persamaan lingkaran berpusat (-2,5), melalui (-1,2)

→ (-1 + 2)² + (2 - 5)² = 10

→ x² + y² + 4x - 10y + 4 + 25 - 10 = 0

→ x² + y² + 4x - 10y + 19 = 0

(x + 2)² + (y - 5)² = r²

→ (3 + 2)² + (0 - 5)² = r²

→ r² = 25

→ (x + 2)² + (y - 5)² = 25

→ x² + y² + 4x - 10y + 4 + 25 - 25 = 0

→ x² + y² + 4x - 10y + 4 = 0

masyarakat : pencurian : hilangnya harta benda masyarakat : mengadakan siskamling.

2. Pertanyaan: Apa pengalaman atau proyek terkait teknologi yang pernah Anda lakukan?

Jawaban: Selama ini, saya telah aktif mengikuti pelatihan dan kursus online untuk mempelajari dasar-dasar pemrograman, jaringan komputer, dan troubleshooting perangkat keras. Saya juga telah mengembangkan beberapa proyek kecil seperti membuat website sederhana dan melakukan konfigurasi jaringan lokal di rumah.

Jawaban: Ketika menghadapi tantangan atau masalah teknis dalam proyek, saya biasanya mulai dengan melakukan penelitian dan mencari solusi melalui sumber daya online, buku, atau berkonsultasi dengan rekan-rekan sejawat.
