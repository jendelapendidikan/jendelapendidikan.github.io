7. barisan 13 + 11 + 9 +.... + Un = - 1800. banyak suku pada deret tersebut adalah...


U1 = a = 13

b = U2-U1 = U3-U2 = 11-13 = 9-11 = -2

Sn = -1800

maka :

Sn = n/2 . (2a+(n-1).b)

-1800 = n/2 . (2.13 + (n-1).(-2))

-1800 = n/2 . (26 + (-2n+2))

-1800 . 2 = n.(26+2-2n)

-3600 = n.(28-2n)

-3600 = 28n-2n²

2n²-28n-3600 = 0

n²-14n-1800 = 0

(n+36).(n-50) = 0

n+36 = 0

n = -36

Harap diperhatikan, karena jumlah suku tidak mungkin negatif, maka jumlah suku yang sebenarnya adalah 50 suku.

n = -36 (tdk memenuhi)  

atau :

n-50 = 0

n = 50 (memenuhi) semoga membantu :-)
